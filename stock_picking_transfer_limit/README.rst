============================
Stock Picking Transfer Limit
============================

This module extends the functionality of picking transfer wizard to
check the transfer quantity. When the quantity is changed in the transfer
line if the new one is greater than the old, the system will raise a warning
and the quantity will be modified to the old quantity.

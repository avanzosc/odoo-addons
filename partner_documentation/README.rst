====================
Parner Documentation
====================

This module adds documentation templates to add specific documentation for partners.

Configuration
=============

#. Go to *Contacts* /Configuration/Partner documentation template
#. Create a new Template.

Usage
=====

#. Access a partner record
#. Use wizard "Import Partner Document"
#. Created documentation lines from templates for customer / supplier / employee

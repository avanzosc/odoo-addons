===============================
Event Track Cancellation Reason
===============================

* New object cancellation reason in event track.
* If the track is cancelled, a wizard to enter the cancellation reasons.
* If the time type field is billable to customer or is paid to employee, an analytic line is created in the same way as if the session had been done.

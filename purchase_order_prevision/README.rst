========================
Purchase order prevision
========================

* In purchase order lines two new dates: "Loading Date" and "Estimated payment
  date".
* The fields "Estimated payment date", it will be calculated with the estimated
  date + the days defined in the payment method of the purchase order. If the
  user enters "Loading Date", this date will be used to perform the
  calculation.

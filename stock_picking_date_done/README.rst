=======================
Stock picking date done
=======================

* In picking new field "Date realized".
* When the picking is "done", and it has the field "Date realized" introduced,
  it sets this date to the "stock.move" and "stock.move.line" objects.
* In pickings new action "Put date realized in picking" to put the date
  realized on pickings in "done" status.

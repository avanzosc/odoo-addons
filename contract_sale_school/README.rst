========================
Contract Sale for School
========================

With this module, when you confirm an order with recurrent product it will create
a contract per product company, payer line and mandate.

If a product is punctual it will create a line according to the months when it
has to be invoiced.

If a product is recurrent it's dates will be taken from the months defined.

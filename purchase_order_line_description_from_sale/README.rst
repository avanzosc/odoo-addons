=========================================
Purchase order line description from sale
=========================================

* When creating a purchase order from a sales order, bring the description of
  the sales line to the description of the purchase line.
* This will be done if the product is defined "Make to order", and in the
  product category the field "Procured pruchase grouping" is not
  "Standard grouping".

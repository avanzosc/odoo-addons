=========================
Helpdesk ticket with task
=========================

* In tickets new field "tasks". If the ticket has a sales order defined, you
  can only select tasks assigned to the sales order lines, or to tasks that are
  not assigned to any sales order line.
* In tasks new shortcut to tickets
